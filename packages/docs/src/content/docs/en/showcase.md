Using ZhuàngTài already? Bring your project to the Showcase. A submission can be a complete application, a library, an SDK, a component, or one representative state module.

## Good showcase candidates

- SDKs, component libraries, or Web Components using Core for framework-independent state.
- Synchronous state relationships in editors, media players, Canvas tools, visualizations, or complex forms.
- Practical use of `useAtom`, Persist, or Sync in React applications.
- Migration stories that explain tradeoffs, benefits, and remaining challenges.

## What to include

- A publicly accessible project or source link.
- The `@zhuangtai-js/*` packages you use.
- The problem ZhuàngTài solves in the project.
- A screenshot, demo URL, or short code excerpt.
- The project name and description you want us to feature.

Post in [Show and tell Discussions](https://github.com/zhuangtai-js/ZhuangTai/discussions/categories/show-and-tell). We will confirm the presentation with you before adding the project to this page.
