The Showcase only includes publicly verifiable real projects. It will not be filled with invented case studies. ZhuàngTài is at the beginning of its adoption phase, so this page first defines the acceptance criteria and submission path.

## What we want to see

- Framework-independent state in an SDK, component library, or Web Component.
- Synchronous state graphs in editors, media players, Canvas, or visualization tools.
- Real use of `useAtom`, Persist, or Sync in React applications.
- Concrete migration tradeoffs from another state library, not merely an API swap.

## How to submit

Share the project link, packages used, why you chose ZhuàngTài, and screenshots or text you permit us to publish in [Show and tell Discussions](https://github.com/zhuangtai-js/ZhuangTai/discussions/categories/show-and-tell). After maintainers verify the project and receive permission, the case can be added to this page.
