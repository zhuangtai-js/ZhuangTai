## Why ZhuàngTài?

<CardGrid>
  <Card
    title="Synchronous by default"
    icon="rocket">
    `set()` updates immediately and `watch()` runs synchronously. There is no hidden batching or
    delayed scheduling.
  </Card>
  <Card
    title="Zero-dependency core"
    icon="puzzle">
    `@zhuangtai-js/core` has no third-party runtime dependencies and keeps the core API small.
  </Card>
  <Card
    title="Composable plugins"
    icon="setting">
    Install plugins with `createAtom()`. The default `atom()` export stays clean and unextended.
  </Card>
  <Card
    title="TypeScript first"
    icon="seti:typescript">
    Written in TypeScript natively, providing complete type inference and type safety.
  </Card>
</CardGrid>

<SplitCard
  title="Install"
  icon="puzzle">
  <div slot="left">
    <p>Install the core package:</p>

```sh
pnpm add @zhuangtai-js/core
```

  </div>
  <div slot="right">
    <p>Install plugins like persistence or React adapter only when you need them:</p>

```sh
pnpm add @zhuangtai-js/core @zhuangtai-js/persist @zhuangtai-js/react
```

  </div>
</SplitCard>
