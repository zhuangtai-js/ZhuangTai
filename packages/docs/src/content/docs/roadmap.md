路线图表达优先级，不是发布日期承诺。Core 的同步契约不会为了增加功能数量而漂移。

## 当前：采用基础

- Vite Vanilla 与 React 示例进入 workspace、类型检查和 CI 生产构建。
- State Lab、诚实对比、集成支持等级和可复现 benchmark。
- Discussions、Issue/PR 模板、贡献指南、安全策略和行为准则。

## 下一步：经过验证的集成

- Next.js App Router fixture，覆盖 hydration、请求隔离和 streaming SSR 边界。
- Astro fixture 与面向库作者/SDK 的示例。
- Node.js 版本矩阵，以及 Bun、Deno 的探索性验证。

## 后续：由真实需求驱动

- 根据早期用户反馈评估 Vue、Svelte、Solid 或 React Native adapter。
- 收集真实项目，建立 [Showcase](/showcase/)。
- 只有在真实采用阻碍明确时才评估 DevTools。

完整双语路线图保存在仓库的 [`ROADMAP.md`](https://github.com/zhuangtai-js/ZhuangTai/blob/main/ROADMAP.md)。建议请提交到 [Discussions](https://github.com/zhuangtai-js/ZhuangTai/discussions)。
