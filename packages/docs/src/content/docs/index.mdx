## 为什么选择 ZhuàngTài？

<CardGrid stagger>
  <Card
    title="同步直达"
    icon="rocket">
    `set()` 立即更新，`watch()` 同步触发，没有隐藏的批处理或延迟调度。
  </Card>
  <Card
    title="零依赖内核"
    icon="puzzle">
    `@zhuangtai-js/core` 没有第三方运行时依赖，核心 API 保持少量且直接。
  </Card>
  <Card
    title="可组合插件"
    icon="setting">
    使用 `createAtom()` 安装插件；默认 `atom()` 保持干净，不受插件影响。
  </Card>
  <Card
    title="TypeScript 优先"
    icon="seti:typescript">
    原生 TypeScript 编写，提供完整的类型推导和类型安全。
  </Card>
</CardGrid>

## 安装

```sh
pnpm add @zhuangtai-js/core
```

需要持久化或 React 适配器时再安装插件：

```sh
pnpm add @zhuangtai-js/core @zhuangtai-js/persist @zhuangtai-js/react
```
