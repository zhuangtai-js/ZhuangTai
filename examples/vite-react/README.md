# Vite React 示例 / Vite React Example

## 中文

展示框架无关的 Core atom 如何通过 `@zhuangtai-js/react` 在 React 组件中读取和更新。

```sh
pnpm install
pnpm --filter @zhuangtai-js/example-vite-react dev
```

## English

Shows how framework-agnostic Core atoms are read and updated in React through `@zhuangtai-js/react`.

```sh
pnpm install
pnpm --filter @zhuangtai-js/example-vite-react dev
```
