# Vite Vanilla 示例 / Vite Vanilla Example

## 中文

展示 `@zhuangtai-js/core` 的 `atom`、`computed` 与同步 `watch` 时间线。

```sh
pnpm install
pnpm --filter @zhuangtai-js/example-vite-vanilla dev
```

## English

Demonstrates `atom`, `computed`, and a synchronous `watch` timeline with `@zhuangtai-js/core`.

```sh
pnpm install
pnpm --filter @zhuangtai-js/example-vite-vanilla dev
```
